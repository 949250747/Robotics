# Robotics
 2020春季学期-机器人学课程程序

包含以下项目：

| 项目       | 内容                                     |
| ---------- | ---------------------------------------- |
| Euler      | 欧拉角与指数矩阵的相互转换，Matlab       |
| Dynamics   | 六轴操作臂正逆运动学，C++                |
| MotionPlan | 六轴操作臂关节空间和笛卡尔空间的轨迹规划 |
